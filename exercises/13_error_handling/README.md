# 錯誤處理

大多數錯誤不會嚴重到需要程式完全停止。有時候，當一個函數發生錯誤時，原因可能很容易理解並能夠應對。例如，如果您嘗試打開一個檔案，但操作失敗是因為該檔案不存在，您可能希望建立該檔案，而不是終止進程。
